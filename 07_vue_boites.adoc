=== Structurizr

Structurizr [.fragment]#- Créé par Simon Brown#

[%step]
--
[source, bash]
----
workspace ... {

    model { ... }
    views { ... }
}
----
--

[.notes]
--
* Simon Brown a créé Structurizr pour résoudre ces deux problèmes
* Domain-Specific Language
* 1 seul workspace par projet
--
